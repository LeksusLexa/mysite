Импорт товаров из Excel

Эта страница нужна для GitHub Pages: клиент заполняет Excel, ты открываешь файл здесь и скачиваешь новый products.json.

Поддерживаются .xlsx, .xls и .csv. Данные не отправляются на сервер, обработка идет в браузере.

Обязательные поля:
sku, name, category, price, image, url
Файл пока не загружен.
